# Instructor Notes
## How to distribute
- Keep this repo as the **template**.
- Each student clicks "Use this template" → creates their own repo.
- Verify Pages is enabled on their fork.

## What to check when grading
1. Topic exists under correct subject.
2. Overview / Approach / Evidence / Reflection all filled.
3. 1–3 screenshots present in `assets/images/`.
4. Site loads without broken links.
5. Reflection ties technical steps to learning outcomes.
